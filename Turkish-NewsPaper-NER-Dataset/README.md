NER dataset from Newpaper sentences. 20.000 sentences are sampled and re-annotated from [Kuzgunlar NER dataset](https://data.mendeley.com/datasets/cdcztymf4k/1).


Data split:

- 18.000 train
- 1000 test
- 1000 dev

Labels:

- CARDINAL
- DATE
- EVENT
- FAC
- GPE
- LANGUAGE
- LAW
- LOC
- MONEY
- NORP
- ORDINAL
- ORG
- PERCENT
- PERSON
- PRODUCT
- QUANTITY
- TIME
- TITLE
- WORK_OF_ART

Annotations are done by [Co-one](https://co-one.co/). Many thanks to them for their contributions.
